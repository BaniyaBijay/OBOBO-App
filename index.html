How to Use

Stopwatch

1. Click "Start" to begin timing.
2. Click "Stop" to pause the stopwatch.
3. Click "Reset" to set the time back to 00:00:00.00.

Lap Timer

1. Click "Start" to begin timing laps.
2. Click "Lap" to record the current lap time.
3. Click "Stop" to pause the lap timer.
4. Click "Reset" to set the lap time back to 00:00:00.00.

Alarm

1. Set the desired time using the input field.
2. Click "Set Alarm" to schedule the alarm.
3. The alarm will sound at the designated time.
4. Click "Cancel Alarm" to stop the alarm.
